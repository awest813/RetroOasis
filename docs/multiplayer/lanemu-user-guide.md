# RetroOasis LAN Rooms User Guide

## Getting Started
RetroOasis LAN Rooms makes playing with friends as easy as joining a lobby. No more manual VPN configuration or firewall headaches.

## Step-by-Step Setup
1. **Open Multiplayer**: Click the "Play Together" icon on the home screen.
2. **Setup Wizard**: If it's your first time, the wizard will guide you through:
   - Checking for Java (OpenJDK).
   - Selecting your `Lanemu.jar` file.
   - Choosing your player name.
3. **Join or Host**:
   - **Host**: Create a new room and share the invite file with your friend.
   - **Join**: Import the invite file shared by your friend.
4. **Test Connection**: Use the "Connection Doctor" to ensure you can see each other.
5. **Launch Game**: Click "Launch" and the emulator will automatically be configured for the session.

## Troubleshooting
- **No Virtual IP**: Ensure LANemu is running and you have allowed it through your firewall.
- **Connection Failed**: Double check that both you and your friend have joined the *same* room access file.
- **Emulator Issues**: Ensure "WLAN" or "Networking" is enabled in your emulator settings.
